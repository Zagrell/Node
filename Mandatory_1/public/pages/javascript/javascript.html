Variables

There are three ways to make variables in javascript. These are var, const, and let

var is seldom used these days, since it is in global scope and are thuss hard to keep track of

let is just a normal variable that you can do whatever you want with

const is a variable that you have to assign on declaration and you cannot reassign it, it is not a constant in the same way Java defines constants, but rather only the pointer is constant

in general:
use const if you can otherwise use let, and do not use var
