Node.js

About

node is a tool used to make it possible to execute javascript code without a browser. This makes it then possible to make server code using javascript

package.json

The package.json is where node has all the metadata for the project, and most importantly for us is where we define dependencies

this here is an example of our super simple package.json where we only say that we want to use express, note that is has to be valid json and be placed in the project's root folder

Installing modules

now that we have our package.json set up we need to install express and its dependencies

to do this we open a terminal in the projects root folder and execute npm install, npm is node's package manager

now all the modules required for our dependencies will be installed in the node_modules folder in the root of the project

Running node

to run a script simply execute node sciptname.js in the terminal.
When we create node project convention is that our entrypoint is called app.js

Live reload

since live reloading is nice we do want it, it is not included in node by default so we have to install a new program. We will be using nodemon

to run your program with live reload simply execute nodemon.cmd app.js in a terminal
